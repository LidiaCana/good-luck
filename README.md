# Good Luck

A raffle platform application where users can organize raffles or participate in them.

## Project Overview

Good Luck enables raffle organizers to create and manage raffles, while participants can browse, subscribe to, and purchase tickets for raffles.

## Documentation

- **[REQUIREMENTS.md](./REQUIREMENTS.md)** - Complete project requirements organized by feature

## Tech Stack

- **Backend**: Phoenix (Elixir)
- **Frontend**: Flutter (Dart)
- **Authentication**: Auth0 or Clerk
- **Payment**: Stripe
- **Database**: PostgreSQL
